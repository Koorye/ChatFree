# Chat Free

一款高度定制化的AI聊天APP，通过免费大语言模型API进行交互。

**本应用是一个用于交作业的DEMO，仅供教育用途，禁止一切商用行为！**

## 快速上手

1. 创建角色

2. 设置API Token

3. 创建对话

## 特性

1. 高度定制化的角色，可以为每个角色设计描述
2. 自定义API，后续将开发更加定制化的设置
3. 对话历史，可以创建多个对话，保存每个对话的历史
4. 角色切换，每个对话中可以切换不同的角色
5. 记忆功能，对话时可选择是否记忆先前的历史，AI将作出不同的回答
